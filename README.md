# queue-management-system

## 📌 Description 
Queue Management System is a web app that lets users join a queue, get a unique number, and view the full queue in real-time. Built with Python (Flask), SQLite, and HTML/CSS, it streamlines waiting lines, ensures fairness, and provides a simple, user-friendly interface for schools, offices, or small businesses.

🚀 Features
- Input student or client name to join the queue
- Automatic queue number generation
- Display the full queue list in real-time
- User-friendly interface for easy navigation

🛠️ Tech Stack
- Python (Flask)
- HTML/CSS
- SQLite
